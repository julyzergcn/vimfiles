vimfiles
========

My custom vimfiles for vim on Windows

Install:
----------

    cd /path/to/vim(e.g. c:\program files\vim)
    git clone https://github.com/julyzergcn/vimfiles.git
    cd vimfiles
    git submodule update --init
    copy _vimrc ..

Update:
----------

    cd /path/to/vim/vimfiles
    git submodule update
